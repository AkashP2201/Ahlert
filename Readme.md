# AHLERT: Proof-of-Concept (PoC)

## Overview

This repository provides a proof-of-concept (PoC) implementation of *AHLERT*, a framework that automatically transforms unstructured Cyber Threat Intelligence (CTI) reports into environment-aware, actionable threat hunting leads.
